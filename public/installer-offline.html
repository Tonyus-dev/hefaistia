A página não executa nada no navegador. Ela apenas mostra comandos e links para baixar os bootstraps públicos.

Você está no celular. A instalação completa é para computador Linux Mint Xfce ou Windows. Abra esta página no computador onde deseja instalar a Kaline Offline.

Problemas comuns

            

              O comando gh não existe
              

                
Instale o GitHub CLI antes de rodar o bootstrap.

                
sudo apt install gh

                
winget install --id GitHub.cli

              

            

            

              Não tenho acesso ao repo
              

                
Use o fluxo interativo do GitHub CLI. Nunca cole tokens nesta página.

                
gh auth login

              

            

            

              Mint bloqueou o script
              

                

                  
cd ~/Downloads
bash kaline-installer-linux-mint.sh

                

              

            

            

              PowerShell bloqueou o script
              

                

                  
powershell -ExecutionPolicy Bypass -File .\kaline-installer-windows.ps1

                

              

            

            

              Ollama não está rodando
              

                Abra ou instale o Ollama. Sem ele, a Kaline pode instalar, mas a IA local ficará
                indisponível até configurar.
              

            

            

              Modelo Ollama ausente
              

                O instalador pergunta antes de baixar modelos. Se você recusar, instale depois com
                ollama pull <modelo>.
              

            

            

              Whisper Small não encontrado
              

                A transcrição ficará indisponível até configurar. Isso não quebra a instalação
                principal.
              

            

            

              Kokoro/Dora não encontrado
              

                A voz local ficará indisponível até configurar. A Kaline ainda pode usar fallback do
                navegador quando aplicável.
              

            

            

              Janelinha não apareceu na barra
              

                Abra o menu de aplicativos, procure “Kaline Janelinha”, clique com o botão direito e
                escolha “Adicionar ao painel”, ou arraste para a barra.
